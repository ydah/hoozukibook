# Hoozuki (鬼灯)

本リポジトリは書籍『Ruby でつくってまなぶ 正規表現エンジン』で解説されているソースコードです。

本リポジトリの各ディレクトリはそれぞれの章に対応しています。

- [ch1/](./ch1/hoozuki) — 『第1章 はじめに ── 正規表現エンジンの役割と本書で作るもの』
- [ch2/](./ch2/hoozuki) — 『第2章 完全一致 ── もっともシンプルなマッチング 』
- [ch3/](./ch3/hoozuki) — 『第3章 リテラル文字の連結 ── AST の導入 』
- [ch4/](./ch4/hoozuki) — 『第4章 選択（|）の実装 ── 複数パターンへの対応 』
- [ch5/](./ch5/hoozuki) — 『第5章 NFA の導入 ── 非決定性有限オートマトン』
- [ch6/](./ch6/hoozuki) — 『第6章 グループ化 ── 括弧による優先順位制御 』
- [ch7/](./ch7/hoozuki) — 『第7章 DFA への変換 ── 決定性オートマトンによる高速化』
- [ch8/](./ch8/hoozuki) — 『第8章 繰り返し（*）── 0 回以上のマッチング』
- [ch9/](./ch9/hoozuki) — 『第9章 その他の繰り返し（+, ?）── 量指定子の拡張』
- [ch10/](./ch10/hoozuki) — 『第10章 エスケープシーケンスと改善のアイデア ── 完成とその先へ』

書籍で解説されていない実験的なコードや最新の変更は[hoozuki](https://github.com/ydah/hoozuki)リポジトリに存在します。

もしプログラム中に問題を見つけた場合は、[Issues](https://github.com/ydah/hoozukibook/issues)に報告をいただけると嬉しいです。
